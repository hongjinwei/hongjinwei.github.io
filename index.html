hexo deploy 时 github pulickey denied的问题

发表于 2016-05-12   |  

#hexo在deploy的时候出错， 提示permission denied

其实是gitgub授权的问题，但是我明明已经在github配置好了我的key，还是出现了这个问题。

看了以下， 貌似github默认看id_rsa， 而我把rsa的名字改了，无法直接识别，用ssh-add来加入到ssd-agent中，但是重启之后又没有了。在mac osx下，可以使用ssh-add -K来把key添加到keychain中，终于解决了。

附上一个segmentfault的回答 link

另外 第一次安装记得需要另外安装deployer：

npm install hexo-deployer-git --save
